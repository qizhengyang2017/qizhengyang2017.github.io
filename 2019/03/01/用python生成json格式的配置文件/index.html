安装制定版本为3.6，为什么python的版本是3.5.5

目录高亮是怎么回事？

说明这个目录权限太高了，所有人都能够访问、修改、删除、运行。
正常情况，应该是Owner可以读写，all可读，root可以修改可执行权限才对。
正常情况，对于目录，应该权限设定为744，对于文件默认权限应该设定为644。
用户私有文件，权限应该设定为600。
